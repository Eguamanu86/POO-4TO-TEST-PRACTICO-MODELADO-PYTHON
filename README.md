# POO 4TO Test Práctico de Modelado en Python

Repositorio con los casos prácticos de Programación Orientada a Objetos para el cuarto test práctico.

## Contenido

- `Caso-01-Sistema-Notas-Academicas.md` - Caso de estudio sobre un sistema de notas académicas.
- `Caso-02-Gestion-Turnos-Atencion.md` - Caso de estudio sobre gestión de turnos de atención.
- `Caso-03-Inventario-Tienda-Colegio.md` - Caso de estudio sobre inventario de tienda escolar.
- `Soluciones-Casos-Practica-Grupos.md` - Soluciones y ejemplo de modelado para los casos.

## Propósito

Este repositorio reúne la documentación de los ejercicios de modelado en Python para la asignatura de POO. Cada caso presenta:

- contexto y objetivo
- análisis y abstracción
- diseño de clases
- requisitos de implementación
- preguntas de comprobación

## Cómo usarlo

1. Abrir el caso en el que quieras trabajar.
2. Revisar el análisis de las entidades y el diseño de clases.
3. Implementar las clases y métodos indicados en Python.
4. Usar `Soluciones-Casos-Practica-Grupos.md` como referencia después de intentar resolverlo.

## Recomendaciones

- Emplea atributos privados con `self._...` y propiedades.
- Mantén las responsabilidades de cada clase bien separadas.
- Añade validaciones básicas según el caso (por ejemplo, no permitir stock negativo o notas inválidas).

## Nota

Este repositorio no incluye código fuente Python directo, solo los casos y su documentación. Puedes crear nuevos archivos `.py` para desarrollar las soluciones localmente.
